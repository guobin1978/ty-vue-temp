# ty指令

## 创建项目
```
ty create <project>
```

## 增加组件
```
ty addcom <name> [-d src/components]
```

## 组件删除
```
ty delcom <name> [-d src/components]
```

## 页面增加
```
ty addpage <name> [-d src/views]
```

## 页面删除
```
ty delpage <name> [-d src/views]
```

## vuex module增加
```
ty addmodule <name> [-d src/store/modules]
```

## vuex module删除
```
ty delmodule <name> [-d src/store/modules]
```

## 生成api(基于yapi)
```
ty createapi
```
